## Getting help

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/gavinc/mapsnatch/issues/new/choose) | Bugs and feature requests for the **CLI** |
| **support@mapsnatch.com** | Account/help with the **hosted** app at [mapsnatch.com](https://mapsnatch.com) |
| **security@mapsnatch.com** | Vulnerabilities (see [SECURITY.md](./SECURITY.md)) |

Please **do not** post API tokens in issues or email.

## Hosted vs CLI

| | CLI (this repo) | Hosted [mapsnatch.com](https://mapsnatch.com) |
|---|-----------------|-----------------------------------------------|
| Cost | Free (AGPL) | One-time paid export |
| Token | Your machine / `.env` | Browser session |
| Support | GitHub Issues | support@mapsnatch.com |
