# MapSnatch CLI

Open-source bulk export for MindMeister using the **public API** — no Business plan required.

```bash
pip install mapsnatch
mapsnatch --dry-run
```

- **Repository:** [github.com/gavinc/mapsnatch](https://github.com/gavinc/mapsnatch)
- **Hosted product:** [mapsnatch.com](https://mapsnatch.com) (browser, one-time payment)
